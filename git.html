Complete Git Commands Reference

Basic Commands

git init - Initializes a new Git repository.
git clone <repository> - Clones an existing repository.
git status - Displays the state of the working directory and staging area.
git add <file> - Stages changes in the specified file.
git commit -m "message" - Commits the staged changes with a message.
git log - Shows the commit history for the current branch.
git diff - Shows changes between commits, commit and working tree, etc.
git branch - Lists all branches in the repository.
git branch <branch-name> - Creates a new branch.
git checkout <branch-name> - Switches to the specified branch.
git merge <branch-name> - Merges the specified branch into the current branch.
git pull - Fetches and merges changes from the remote repository.
git push - Uploads local commits to the remote repository.
git remote -v - Lists remote repositories.
git remote add <name> <url> - Adds a new remote repository.
git fetch - Downloads changes from the remote repository without merging.
git reset <file> - Unstages a file, reverting it back to the last committed state.
git stash - Temporarily saves changes that are not ready to be committed.
git stash pop - Restores the most recently stashed changes.
git tag <tag-name> - Creates a tag for a specific commit.

Advanced Commands

git rebase <branch> - Reapplies commits from the current branch on top of another branch.
git cherry-pick <commit> - Applies the changes from a specific commit to the current branch.
git revert <commit> - Creates a new commit that undoes the changes from a specific commit.
git clean - Removes untracked files and directories.
git submodule - Manages submodules (repositories within a repository).
git filter-branch - Rewrites the commit history.
git bisect - Finds the commit that introduced a bug.
git worktree - Manages multiple working trees.

Gitignore

git ignore <pattern> - Ignores files or directories matching the specified pattern.
git ignore --global <pattern> - Ignores files or directories globally.

Git Hooks

git hooks - Custom scripts that run at specific points in the Git workflow.

Git Configuration Commands

git config <key> <value> - Sets a configuration option.
git config --global <key> <value> - Sets a global configuration option.
git config --global --get-all - Displays all globally configured options.
git config --global --get-all alias - Displays all globally configured aliases.
git config --global --get-all core - Displays all globally configured core settings.
git config --global --get-all credential - Displays all globally configured credential settings.
git config --global --get-all safe.directory - Displays all globally configured safe directories.
git config --global --get-all user - Displays all globally configured user settings.
